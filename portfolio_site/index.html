Sobre

Olá, sou Danilo, um estudante apaixonado por tecnologia em busca de oportunidade de estágio em dados. Tenho habilidades básicas em ferramentas de nuvem, como Oracle e Azure, que pude aprimorar durante projetos práticos em gerenciamento de dados e infraestrutura de TI, incluindo criação de VMs, bancos de dados e redes completas com Web Server e banco de dados privado. Além disso, desenvolvi habilidades em Python e SQL através de projetos na faculdade, onde criei bancos de dados com modelagem, objetos, população e manipulação de dados. Também realizei atividades práticas em Python para análise, exploração de dados e modelos de machine learning. Como jovem aprendiz, atuei na área de estratégias operacionais, mantendo relatórios, criando dashboards e mantendo contato com a engenharia de dados, aprendendo sobre a dinâmica de empresas com cultura data driven. Ademais, fui estudante de um projeto social da USP voltado para a capacitação tecnológica, onde pude aprender diversos temas relevantes como economia com TI, desenvolvimento web, inovação tecnológica, arquitetura tcp/ip, big data, empreendedorismo e negócios, fundamentos em linguagem de programação, orientação de carreiras, matemática financeira, entre outros.
